# SETUP GUIDE

## Prerequisites
- No special dependencies are required to run this project locally.
- A modern web browser (Chrome, Firefox, Safari, Edge).

## Local Development
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd valentine26
   ```

2. Open `index.html` in your favorite web browser.
   - Alternatively, you can use a simple local server if you'd like:
     ```bash
     npx serve .
     ```

## Customization
- **Change the question**: Edit the `<h1>` tag in `index.html`.
- **Change the colors**: Modify the CSS variables in `style.css`.
- **Change the GIF**: Update the success GIF URL in `script.js`.
